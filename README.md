# 校园网络 ENSP
## 项目简介
基于华为eNSP搭建三层架构园区校园网，模拟教学楼、宿舍、办公区、服务器区多业务场景，完成网络隔离、动态路由、无线覆盖、安全策略、外网接入全套部署，完整复刻企业级校园网实施流程。

## 技术栈
华为VRP、eNSP仿真、VLAN、Trunk、STP、OSPF动态路由、DHCP全局地址池、ACL访问控制、NAT Easy IP、AC+Fit AP无线组网

## 实现功能
1. 三层分层架构：核心层-汇聚层-接入层，规范化IP网段规划
2. VLAN广播域隔离，STP防止二层环路
3. OSPF全网路由自动收敛，跨VLAN互通
4. DHCP自动分配终端IP地址，简化管理
5. ACL安全策略：限制学生区访问办公内网服务器
6. NAT实现内网统一访问互联网
7. AC管理无线AP，校园全覆盖WiFi
8. 全流程连通性、路由、安全策略测试验证

## 仓库资源目录
- 校园网/：eNSP工程拓扑文件（.topo）
- 配置/：交换机、路由器、AC完整配置脚本
- 实验报告.docx：需求分析、拓扑设计、排错记录、测试结果

## 运行说明
1. 本地部署eNSP及配套VRP镜像
2. 导入`校园网/`下拓扑文件启动设备
3. 导入对应设备cfg配置文件
4. PC终端执行ping、tracert验证网络连通性
